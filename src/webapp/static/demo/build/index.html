CloudBrain

http://cloudbrain.rocks

Real-time EEG data analysis and visualization

  • 1. Stream EEG data into a central universally accessible database
  • 2. Detect patterns by analyzing streamed-in EEG data in real-time
  • 3. Trigger actions associated with these patterns
  • 4. Visualize data and patterns in real-time

Cloudbrain Building Blocks

Stream anything

  • CloudBrain works with OpenBCI
  • You can connect other wearables

API endpoints

Get metrics

# get metrics
